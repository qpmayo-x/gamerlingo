━━━ GamerLingo가 다른 이유 ━━━

✅ 게임 슬랭을 이해합니다

• Discord — 모든 채팅 메시지에 🌐 버튼으로 즉시 번역
• Twitch — 라이브 채팅을 원클릭 번역
• Steam — 리뷰, 포럼, 커뮤니티 게시글 번역
• 모든 사이트 — 텍스트 선택 후 우클릭으로 번역

✅ 자연스러운 번역
단어별 기계적 번역이 아닙니다. 실제 게이머가 말하는 것처럼 전체 문장을 번역합니다.

예시:
• 입력: "GG EZ noob get rekt"
• 구글 번역: "GG 쉬운 초보 파괴"
• GamerLingo: "지지 ㅋㅋ 너무 쉬웠다 뉴비 박살났네ㅋㅋ"

영어, 일본어, 중국어, 한국어, 스페인어, 포르투갈어, 러시아어, 독일어, 프랑스어, 힌디어 — 모든 언어 쌍에 슬랭 사전 탑재.
