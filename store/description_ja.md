# Chrome Web Store リスティング（日本語）

## 名前
GamerLingo - AIゲーム翻訳

## 短い説明（132文字以内）
ゲーマー専用AI翻訳。GG、noob、nerfなど386以上のスラングを自然に翻訳。Discord・Steam・Twitchに対応。10言語サポート。

## 説明

🎮 GamerLingo — ゲーマーの言葉がわかる唯一の翻訳ツール

Google翻訳が「GG EZ noob get rekt」をめちゃくちゃに訳すのにうんざりしていませんか？GamerLingoはゲームコミュニティのために作られたAI翻訳ツールです。

━━━ GamerLingoが違う理由 ━━━

✅ ゲーミングスラングを理解
「nerf」がナーフガンではなく弱体化を意味することを知っています。「clutch」「inting」「草」など386以上のスラングを自然に翻訳。

✅ ゲームする場所で使える
• Discord — チャットメッセージに🌐ボタンが表示、ワンクリック翻訳
• Twitch — ライブチャットをワンクリックで翻訳
• Steam — レビュー、フォーラム、コミュニティ投稿を翻訳
• 任意のサイト — テキストを選択して右クリックで翻訳

✅ 自然な翻訳
単語ごとのロボット的な出力ではありません。実際のゲーマーが言うように翻訳します。

例：
• 入力: "GG EZ noob get rekt"
• Google翻訳: "GG イージー noob ボコボコにされた"
• GamerLingo: "おつ、楽勝だったわ　雑魚すぎボコボコだなｗ"

✅ 10言語対応
英語・日本語・中国語・韓国語・スペイン語・ポルトガル語・ロシア語・ドイツ語・フランス語・ヒンディー語 — 全言語ペアにスラング辞書搭載。

✅ 超高速
翻訳は2秒以内に表示。キャッシュ済み結果は即時表示。
