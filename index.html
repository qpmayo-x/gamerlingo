GamerLingo

ゲーマーの言葉がわかるAI翻訳ツール

GamerLingoとは？

GamerLingoは、ゲームチャットをAIで翻訳するChrome拡張機能です。「GG EZ」「noob」「rekt」などのゲーミングスラングを自然に翻訳します。Discord、Steam、Twitchで直接使えます。

主な機能

  • 80以上のゲーミングスラングに対応したAI翻訳
  • 10言語対応：日本語、英語、中国語、韓国語、スペイン語、ポルトガル語、ロシア語、ドイツ語、フランス語、ヒンディー語
  • Discord、Steam、Twitchのチャットをワンクリックで翻訳
  • 右クリックメニューから翻訳
  • ポップアップからのクイック翻訳
Chrome Web Storeで入手

使い方

  • Chrome Web Storeから拡張機能をインストール
  • Discord、Steam、Twitchでテキストを選択し、右クリックで翻訳
  • またはチャットメッセージに表示される翻訳ボタンをクリック
  • 拡張機能のポップアップからProプランにアップグレード
