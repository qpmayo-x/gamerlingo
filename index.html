What is GamerLingo?

GamerLingo is a Chrome extension that translates gaming chat using AI. It understands gaming slang like "GG EZ", "noob", "rekt" and translates them naturally — not like Google Translate. Works directly on Discord, Steam, and Twitch.

Features

  • AI-powered translation that understands 80+ gaming slang terms
  • 10 languages: English, Japanese, Chinese, Korean, Spanish, Portuguese, Russian, German, French, Hindi
  • One-click translate on Discord, Steam, and Twitch chat
  • Right-click context menu translation
  • Quick translate from the popup

How it works

  • Install the extension from the Chrome Web Store
  • Select text on Discord, Steam, or Twitch and right-click to translate
